# Architektur-Entscheidungen (ADR)

Hier werden wichtige technische Entscheidungen dokumentiert — mit Begründung.
Das verhindert dass man in 6 Monaten nochmal dieselbe Diskussion führt.

## Format

Jede Entscheidung beantwortet:
- **Was** wurde entschieden
- **Warum** (Kontext, Alternativen die verworfen wurden)
- **Konsequenzen** (Trade-offs)
